Tutorial of HTML

HTML, which stands for HyperText Markup Language, is the standard language used to create web pages. HTML is used to create the structure and content of a web page, including headings, paragraphs, images, links, and more.

To start creating your own HTML page, you will need a text editor like Notepad, Sublime Text, or Visual Studio Code. Once you have a text editor, you can start coding!

Here are the basic steps to create an HTML page:

Open your text editor and create a new file.

Type the following code:

HTML Basics

          <!DOCTYPE html>
          <html>
            <head>
              <title>Your Title Here</title>
            </head>
            <body>
          
            </body>
          </html>

This code sets up the basic structure of your web page. The <!DOCTYPE html> tag tells the browser that this is an HTML document. The <html> tag wraps around the entire document, while the <head> and <body> tags divide the document into two parts. The <title> tag sets the title of your web page, which will be displayed in the browser's title bar. Now it's time to add some content to your page. Within the <body> tags, you can add any of the following HTML tags to structure your content:

<h1> to <h6> for headings

<p> for paragraphs

<a> for links

<img> for images

<ul> and <li> for unordered lists

<ol> and <li> for ordered lists

Here's an example of how you can add a heading, paragraph, and image to your page:

    <html>
    <body>
      <h1>Welcome to my Web Page</h1>
      <p>This is my first web page. I'm excited to learn more about HTML!</p>
      <img src="your-image-url-here" alt="Image Description">
    </body>
  </html>

Save your file with a .html extension, such as "index.html".

Open your web browser and navigate to the file you just created. You can do this by selecting "Open File" from the browser's menu and selecting your file.

Voila! You've created your first HTML web page.

Of course, this is just a basic overview of HTML. There are many more HTML tags and attributes you can use to create complex web pages. I recommend checking out some online resources, such as w3schools.com, to learn more.
